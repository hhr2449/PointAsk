# PointAsk Beta Installation

1. Obtain `pointask-mvp-0.1.0.zip` and extract it locally.
2. Open `chrome://extensions`, enable Developer mode, and choose **Load unpacked**.
3. Select the extracted directory containing `manifest.json`.
4. Open or refresh `https://chatgpt.com/`.
5. Use **Details → Extension options** to configure prompt mode, new-thread expansion, pending expiry, local metrics export, or data clearing.

This beta is not published in the Chrome Web Store. Chrome may display a developer-mode warning after restart.
