# PointAsk Beta Feedback Guide

Please focus feedback on:

- whether selecting text naturally suggests a local follow-up;
- whether copying and opening a new chat feels too cumbersome;
- whether it is clear that the user must paste and send manually;
- whether manual answer selection and confirmation are understood;
- whether inline collapse improves readability;
- whether a second local round feels useful;
- whether the no-auto-send/no-auto-scrape boundary is acceptable;
- whether anchors restore reliably after refresh and navigation;
- which step most often interrupts the flow;
- whether you would continue using PointAsk.

Do not include private conversation text, account identifiers, cookies, or screenshots containing sensitive data. The optional metrics export contains counters only.
